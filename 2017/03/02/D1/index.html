Ghost子站博文简汇

发表于 2017-03-02   |   分类于 杂录 ， 站务

鉴于Hexo主站纯VPS化后，发布与更新的操作已可随时随地进行，加上精力有限，对Ghost的“魅力”也已有所体验，因此决定基于Ghost构建的子站（hello站）暂停更新。为便于今后回顾，将于此子站已发布的几篇博文汇总于下，以便回顾与归档（两篇与主站并发的博文仅列标题）。

停止折腾，回归宁静丨170201

随心所欲开写吧…丨170202

作为许久以来建站方面的一个心结，也作为此前数日折腾的重要妥协成果之一，搭建在Ghost+Docker+Vultr的这个子博客算是“开博”了。究竟应该在这上面写点什么呢？毫无疑问，必须与基于Hexo的主站相区隔，不能再为自己添麻烦，做些无用功，搭建个与主站同步更新的“镜像站”之类的了。

想了一下，还是不做严格限定，就随心所欲的写点不登大雅的纯随笔、纯杂录之类的吧，不讲求文法、不讲求工整对仗、不必字斟句酌、不强求保持更新频度、不……（当然，满意之作也可入选主站发布）

标签（分类）之类的也简化点，就分两类，代号A为随笔类，代号D为杂录类（遵照主站排序习惯）；鉴于容器化的稳定性，切记随时一键导出备份。

OK，这就开始吧！

对VPS的一些收官调整丨170204

就在刚才，刚刚完成了对VPS的一次收官调整，下面简单列举下这最后一次如此熬夜（但愿是）的战果——

  • 主机面板由收费的AMH改为免费的BT（宝塔）
  • 由互抢端口的A&N并存改为清净的仅nginx独存（wp架构的麦收客镜像站因此姑且取消了“养眼”的固定链接结构）
  • 受此益，ghost子站终可通过不含端口的真独立域名访问（子站顺势正式定名及定域名，可简称hello站，庆贺一下～）

OK，切记：架构已清，别再这么折腾了，别再这么熬夜了！！

正式开篇语丨170204

新的子博客（子站）算是开写了，不仅域名与站名已经确定，而且“自由自在，所思所想”的立足点也已确定，那就没必要多说什么了，这就随时随地开始写吧～

仅以此，作为此ghost小站的正式开篇语。

主机“失联”之善后丨170204

因Vultr主机莫名其妙“失联”，今晚被迫进行了一次重启，不出所料，这个并不完善的ghost小站不得不重新组建了下。虽然占用时间和精力并不多（得益于Docker架构及充足备份），但因移动端书写后无法进行导出备份，如果此类故障出现在移动端书写后尚未通过电脑端备份的情况下，那无疑会导致新文字的丢失，那可就杯具了～

因此，是要考虑在移动端畅快书写后的及时容灾备份的问题了，毕竟，未雨绸缪要远胜于亡羊补牢啊！！

PS：可喜的是，基于Wordpress的旧站已成功完成了在nignx下的正常固定链接显示，感谢那篇清晰易懂的博文。

转站“主机公园”丨170205

年付购买需谨慎，国外绑卡需远离丨170207

年付VPN的退款还未有结果，在Vultr中试图删除绑定信用卡失误又从aya的账户中划走了10$，本来通过主机公园一台VPS就可以全搞定的事情，费了半天周折却还要赔了夫人又折兵，呜呜~~

虽均已申请退款，目前尚无结果，但也不做更多期待，实在不济就算拿这二百多元交个学费、买个教训吧！

今后切记：年付购买需谨慎，国外绑卡需远离！！
